# Python Mini Projects for Students

This repository contains beginner-friendly Python mini projects that help students improve programming skills through practical examples.

## Projects Included

1. Mad Libs Generator
2. Random Password Generator
3. Email Slicer
4. Number Guessing Game
5. Tic-Tac-Toe Game

## Technologies Used

- Python
- Random module
- String operations

## How to Run

Clone the repository

git clone https://github.com/srikanthymts/python-mini-projects-for-students

Navigate to project folder

cd password-generator

Run the script

python password_generator.py

## Purpose

These mini projects help students understand:

- loops
- conditions
- functions
- data structures
- basic programming logic
